🔋 卡尔曼滤波法估算SOC — 技术调研报告

调研时间：2026年6月  |  适用领域：动力电池/储能BMS  |  作者：WorkBuddy AI

1. 概述

荷电状态（State of Charge, SOC）是电池管理系统（BMS）最核心的状态参数，直接关系到电池的安全运行、寿命管理和能量调度。SOC无法通过传感器直接测量，只能通过电压、电流和温度等外部可观测信号进行间接估计。

卡尔曼滤波（Kalman Filter, KF）及其衍生算法是当前工业界应用最广泛的高精度SOC估计方法。它以电池等效电路模型为基础，结合安时积分法的递推预测与端电压的观测修正，在递推过程中实现"预测-更新"的闭环迭代，有效抑制传感器噪声和初始误差的影响。

📌 核心结论

行业主流方案为 "安时积分（AH） + 扩展卡尔曼滤波（EKF）" 组合算法：安时积分负责实时递推，EKF通过等效电路模型和OCV曲线进行状态观测与修正。实测相对误差可控制在 2%~5%，满足GB/T 38661及《电动汽车用电池管理系统技术条件》等国标对SOC精度的要求。

2. SOC定义与估算挑战

2.1 SOC数学定义

SOC(t) = SOC(t₀) − (1 / Qmax) · ∫t₀t η · I(τ) dτ
符号含义单位
Qmax电池当前最大可用容量（含老化修正，= 额定容量 × SOH）Ah
I(τ)充放电电流（充电为正/放电为正，符号约定依系统）A
η库伦效率（充放电能量转换效率）—
SOC(t₀)初始SOC值%

2.2 SOC估算的核心难点

难点说明
不可直接测量SOC是隐含状态量，只能通过电压/电流间接推断，本质是"状态观测"问题
初始值未知上电时SOC初值往往未知（如更换电池、长时间静置后），需要算法快速收敛
传感器噪声电流/电压传感器存在测量噪声和漂移，安时积分会累积误差（开环积分）
电池非线性OCV-SOC关系高度非线性，且受温度、老化(SOH)、倍率等因素耦合影响
LFP电压平台磷酸铁锂电池约70% SOC区间内电压几乎不变，OCV法几乎失效
时变参数电池内阻、极化电容等参数随温度、SOC、SOH动态变化

3. 卡尔曼滤波理论基础

3.1 核心思想

卡尔曼滤波是一种最优递归状态估计器，核心思想是：利用系统的状态空间模型，在每个采样时刻交替执行"时间更新（预测）"和"测量更新（校正）"两个步骤，以最小均方误差为准则，融合模型预测值与传感器测量值，得到状态的最优估计。

先验估计
x̂ₖ⁻ = A·x̂ₖ₋₁ + B·uₖ
→
先验协方差
Pₖ⁻ = A·Pₖ₋₁·Aᵀ + Q
→
卡尔曼增益
Kₖ = Pₖ⁻·Hᵀ·(H·Pₖ⁻·Hᵀ+R)⁻¹
→
后验估计
x̂ₖ = x̂ₖ⁻ + Kₖ·(zₖ−H·x̂ₖ⁻)
→
后验协方差
Pₖ = (I−Kₖ·H)·Pₖ⁻

3.2 五大核心方程

预测阶段（时间更新）

状态预测：   x̂k− = A·x̂k−1 + B·uk
协方差预测：   Pk− = A·Pk−1·AT + Q

校正阶段（测量更新）

卡尔曼增益：   Kk = Pk−·HT·(H·Pk−·HT + R)−1
状态更新：   x̂k = x̂k− + Kk·(zk − H·x̂k−)
协方差更新：   Pk = (I − Kk·H)·Pk−
符号含义
x状态向量（含SOC）
A状态转移矩阵
B控制输入矩阵
u控制输入（电流I）
z测量值（端电压Ut）
H观测矩阵
Q过程噪声协方差矩阵
R测量噪声协方差矩阵
P状态估计误差协方差
K卡尔曼增益

3.3 为什么适用于SOC估算

关键优势：KF能同时处理"模型预测"（安时积分递推）和"传感器观测"（端电压反推SOC），通过卡尔曼增益K动态权衡两者的信任度。当模型预测不确定性大（P大）时，更相信测量值；当测量噪声大（R大）时，更相信模型预测。

4. 电池等效电路模型

等效电路模型（ECM）是卡尔曼滤波SOC估计的模型基础，将电池的电化学动态特性用电阻、电容和电压源组成的电路网络来近似描述。

4.1 主流等效电路模型对比

模型RC阶数精度计算量适用场景
Rint模型 0阶 低 极小 仅用于简单仿真，不能用于SOC估计
Thevenin模型（一阶RC） 1阶 中 小 对精度要求不高的场景，嵌入式算力受限时
PNGV / 二阶RC模型 2阶 高 中 行业主流方案，精度与实时性的最佳平衡
三阶RC模型 3阶 更高 较大 高端应用，对瞬态响应要求极高的场景
⚡ 工程实践结论：二阶RC（PNGV）模型是综合推荐方案。一阶模型精度不足，三阶及以上模型虽然精度略有提升，但计算复杂度大幅度增加，且模型过拟合风险上升，不利于嵌入式实时运行。二阶模型在精度（<20mV电压跟踪误差）和实时性之间取得最优平衡。

4.2 二阶RC等效电路模型结构

模型参数

参数物理含义典型值范围(314Ah LFP)
Uocv(SOC)开路电压（SOC的函数）2.5~3.65 V
R0欧姆内阻（瞬时电压响应）0.2~1.5 mΩ
R1, C1第一RC环节（电化学极化，秒级时间常数）R₁:0.1~0.5mΩ, τ₁:几秒~几十秒
R2, C2第二RC环节（浓差极化，分钟级时间常数）R₂:0.05~0.3mΩ, τ₂:几十秒~数分钟

4.3 状态空间方程建立

以二阶RC模型为例，离散化后的状态空间方程：

状态方程（预测）：
[SOCk+1, U1,k+1, U2,k+1]T = A·[SOCk, U1,k, U2,k]T + B·Ik + wk
观测方程（校正）：
Ut,k = Uocv(SOCk) − U1,k − U2,k − Ik·R0 + vk

其中 U1, U2 为两个RC环节的极化电压，wk为过程噪声，vk为测量噪声。

5. 卡尔曼滤波族算法详解

电池SOC估计问题中，OCV-SOC关系是强非线性的，因此经典线性KF不能直接使用，需要采用非线性扩展版本。以下按复杂度递增顺序介绍六种主流算法。

5.1 经典卡尔曼滤波（KF）

项目说明
适用条件系统为线性，状态转移和观测函数均为线性变换
SOC应用不直接适用 — OCV-SOC关系非线性，需先线性化处理
地位理论基础，所有衍生算法的基础框架

5.2 扩展卡尔曼滤波（EKF） ★ 行业主流

核心思想

通过对非线性函数在当前估计点进行一阶泰勒展开（计算雅可比矩阵），将非线性系统局部线性化，然后应用标准KF框架。

处理非线性OCV的方法

在每一步计算中，用当前SOC估计值对OCV-SOC曲线求导得到局部斜率（dUocv/dSOC），用该斜率替代线性观测矩阵中的对应元素。这相当于在当前工作点把非线性OCV曲线近似为切线。

项目说明
非线性处理一阶泰勒展开（计算雅可比矩阵 Jf, Jh）
精度良好，稳态误差 <3%（搭配二阶RC模型）
计算复杂度中等（每步需计算雅可比矩阵）
收敛速度较快，对初值不敏感（通常几十秒~几分钟可收敛到真实值）
主要缺陷① 强非线性时线性化误差大；② 雅可比矩阵计算可能数值不稳定；③ 噪声协方差Q/R固定，不适应工况变化
工程地位当前车载BMS和储能BMS中使用最广泛的算法
实际案例：某BMS实测数据：EKF计算结果SOC₁=47.2%，真值（OCV静置法）SOC=48.3%，相对误差仅2.3%。

5.3 无迹卡尔曼滤波（UKF）

核心思想

不进行线性化近似，而是通过无迹变换（Unscented Transform）：在状态均值周围选取一组确定性采样点（Sigma点），将这些点直接通过非线性函数传播，然后从变换后的点集重构统计特性（均值和协方差）。

项目说明
非线性处理无迹变换（UT），精度可达二阶（泰勒展开意义上），部分情况下三阶
精度优于EKF，尤其在高非线性区域（如LFP电压平台两端）
计算复杂度高于EKF（n维状态需要2n+1个Sigma点的传播，实际约EKF的2~3倍）
雅可比矩阵无需计算 — 免除雅可比推导和编程工作量
主要缺陷① 计算量较大（2n+1次非线性传播）；② 高维时可能出现非正定协方差
适用场景精度要求高、算力充足的中高端BMS平台
EKF vs UKF 关键差异：EKF通过局部线性化近似非线性函数；UKF通过采样点传播近似概率分布。研究表明，UKF在LFP电池SOC估计中的精度通常比EKF提高约0.5%~1.5%，尤其在OCV曲线平坦区（LFP的3.28~3.33V平台）表现显著优于EKF。

5.4 自适应扩展卡尔曼滤波（AEKF）

核心思想

标准EKF假设噪声统计特性（Q和R）固定不变，但实际电池运行中，传感器噪声水平、模型误差会随温度、SOC、老化等因素变化。AEKF通过在线估计噪声协方差，使得滤波器能自适应匹配实际工况。

项目说明
自适应机制① 新息协方差匹配法（Innovation Covariance Matching）；② 极大似然估计法（MLE）；③ 变分贝叶斯（VB）近似
精度动态工况下优于固定噪声EKF，尤其温度变化剧烈或老化后
鲁棒性显著优于EKF，能抵抗噪声统计特性变化引起的发散
计算量比EKF略高（增加滑动窗口协方差计算，约增加15%~30%）
主要缺陷窗口长度选择影响自适应性；协方差估计可能不稳定

5.5 双扩展卡尔曼滤波（DEKF）

核心思想

使用两个并行运行的EKF——一个估计状态（SOC），另一个估计模型参数（如内阻R₀、容量Qmax），实现SOC-SOH联合在线估计。两个滤波器交替迭代：参数滤波器用当前状态估计更新参数，状态滤波器用更新后的参数重新估计状态。

项目说明
估计目标同步在线估计SOC + SOH（容量衰减/内阻增长）
双时间尺度SOC估计（秒级/毫秒级）+ SOH/参数估计（分钟级/小时级）— 大幅降低算力需求
精度老化场景下优于单EKF，SOC误差可保持在3%以内（即使容量衰减至80% SOH）
计算复杂度中等偏高（两个EKF + 参数辨识），但多时间尺度策略可优化
主要缺陷① 两个滤波器耦合可能相互干扰；② 多参数联合估计的收敛性需仔细调参
前沿方向结合粒子群优化(PSO)的ADEKF，进一步提升SOC-SOH联合估计精度

5.6 容积卡尔曼滤波（CKF）

项目说明
原理基于三阶球面-径向容积规则，使用2n个等权重容积点进行非线性传播
精度理论精度等同UKF（三阶），但数值稳定性更好
优点① 严格数学推导保证协方差正定性；② 采样点数量少（2n vs UKF的2n+1）
缺点嵌入式实现经验较少，工程成熟度低于EKF/UKF
发展平方根CKF(SCKF)进一步提升数值稳定性，正在成为学术研究热点

7. 工程实现要点

7.1 电池模型参数辨识（HPPC测试）

参数辨识是卡尔曼滤波SOC估计的前提。行业标准方法为HPPC（Hybrid Pulse Power Characterization）测试：

HPPC实验流程（参照FreedomCAR规范）

  1. 在不同SOC点（每5%或10% SOC间隔）执行脉冲充放电
  2. 脉冲放电：1倍 Imax_dch，持续10~30秒
  3. 静置40秒，脉冲充电：0.75倍 Imax_ch
  4. 从电压响应曲线中提取R₀（瞬时电压跳变）、R₁C₁（快速极化）、R₂C₂（慢速极化）
  5. 采用最小二乘法（LS）或递推最小二乘法（RLS）进行参数拟合
  6. 多维度测试：不同温度（−20℃/0℃/25℃/40℃）+不同SOH状态（100%/90%/80%）
关键提示：参数辨识的准确性直接影响SOC估计精度。建议采用带遗忘因子的递推最小二乘法（FFRLS）实现在线参数更新，以应对电池老化引起的参数漂移。

7.2 SOC-OCV曲线获取

步骤说明
1. 基础测试室温（22±3℃）下，以1C电流充电至SOC每变化5%，静置3小时，记录稳态OCV
2. 多维修正在不同温度(-20/0/10/25/40℃)和不同SOH(100%/90%/80%)下重复测试
3. 充放电磁滞充电OCV和放电OCV存在滞回电压差（LFP约5~15mV），需单独考虑
4. 曲线拟合常用多项式拟合（6~8阶）或分段线性插值，确保平滑且单调递增

7.3 噪声协方差矩阵整定

Q和R的经验整定原则

参数物理含义整定依据影响
Q（过程噪声）模型不确定性（电流测量误差+容量误差+参数误差）基于电流传感器精度和容量标定误差Q大→更信任测量值（响应更快但波动大）
R（测量噪声）电压测量不确定性（传感器噪声+模型电压预测误差）基于电压传感器精度和模型残差统计R大→更信任模型（更平滑但响应慢）
⚠️ Q/R建议保持同一数量级，避免滤波器"顾此失彼"。实际工程中，通常先通过静态测试确定R（传感器数据手册或离线统计），再通过动态工况调优Q，直至新息序列呈现白噪声特性。

7.4 初值问题与收敛性

要点说明
初值不敏感卡尔曼滤波是递推算法，对SOC初始猜测不十分敏感，可在运行中自动收敛
加速收敛可结合静置OCV查表获得较好初值，或在EKF初始阶段设置较大的初始协方差P₀
状态初值除SOC外的极化电压U₁、U₂初值通常设为0
典型收敛时间从随机初值收敛到真值±3%以内，通常需要数分钟至十几分钟（取决于工况）

8. LFP磷酸铁锂电池的特殊考量

LFP电池SOC估计的特殊难点

磷酸铁锂（LFP）电池的OCV-SOC曲线在中段（约20%~80% SOC）存在极长的电压平台（约3.28~3.33V），电压变化极小（<10mV/10%SOC），导致dUocv/dSOC ≈ 0。在此区间内：

  • EKF的线性化误差急剧增大 — 一阶泰勒展开的局部斜率几乎为0，观测方程提供的校正信息极少
  • OCV法几乎失效 — 无法从端电压有效推断SOC
  • 卡尔曼增益K极小 — 滤波器实际上"拒绝"测量更新，纯依赖安时积分递推

针对LFP的改进策略

策略方法效果
UKF替代EKF无迹变换处理非线性，不依赖局部导数平坦区精度提升0.5%~1.5%
滞回模型补偿对充放电方向分别建立OCV曲线减少滞回引起的系统误差（~1%~2% SOC）
多算法融合电压平台区：AH积分为主 + 动态限制；两端陡峭区：EKF/UKF充分修正全SOC区间精度均衡
定期满充满放校准利用满充（3.65V）或满放（2.5V）时的确定OCV校准SOC周期性消除累积误差
EIS辅助利用电化学阻抗谱（EIS）特征参数辅助判断SOC区间前沿方向，工程落地需进一步验证
针对314Ah LFP电芯的建议：鉴于该电芯OCV平台极为平坦（磷酸铁锂本质特性），建议优先采用AEKF或UKF + 二阶RC模型方案，并配备定期满充校准机制。若条件允许，可探索结合EIS特征的分区自适应策略。

9. 储能系统中的应用特点

维度动力电池BMS储能BMS对卡尔曼滤波算法的影响
工况动态性剧烈变化（加速/制动）相对平缓（恒功率充放居多）储能工况下模型预测更准确，EKF跟踪压力更小
SOC精度要求±5%±3%（部分场景±2%）储能对SOC精度要求更高（影响调度和收益）
算力资源受限（车规MCU）较充裕（边缘网关/工控机）储能可使用更复杂算法（UKF/AEKF/DEKF）
电芯数量数百颗串联数千~数万颗簇级/堆级SOC融合是新增挑战
运行时长间歇（日行/夜停）连续（24h不间断）储能更依赖长期漂移抑制和SOH联合估计
校准机会每次充满取决于调度策略储能不一定每天满充满放，校准机会更少
储能BMS SOC估计的特殊需求：在大规模储能系统中，除了单电芯SOC估计，还需考虑簇级SOC均衡、多PACK并联的SOC融合以及SOC-SOH联合老化追踪。DEKF（双扩展卡尔曼滤波）在这些场景下具有天然优势。
